Tipo de Ataque Descrição
Phishing Ataque de engenharia social em que os atacantes tentam obter informações pessoais enganando as vítimas.
Malware Software malicioso projetado para danificar sistemas, roubar informações ou causar outros danos.
DDoS Ataque de negação de serviço distribuído que sobrecarrega um servidor, tornando-o inacessível.
Ransomware Software malicioso que criptografa os arquivos da vítima e exige um resgate para desbloqueá-los.
SQL Injection Ataque em que os atacantes inserem comandos SQL maliciosos em campos de entrada de dados para acessar um banco de dados.
